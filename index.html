The BI data locator for dummies

The BI data locator for dummies helps defining BI data by telling you from where the data is extracted, how it is defined and if it can be cross referenced in Tableau and/or business Objects for business analysis. This BI data is limited to print circulation and the target groups are employees outside of the core business analytics team to encourage autonomy in business operations. The business analysis is meant to be executed in existing tools and knowledge of how to build analysis platforms should not be required.

BI Parameter

The BI parameter is the name of the parameter you find in your BI analysis tool. This is the core dimension of your business analysis.

BI Parameter sub group

This is the sub group of parameters of which one can choose from. E.g. Units such as delivered subscription units per day or characteristics such as package type or customer type.

Source of parameter data

This defines from which data table the information is extracted from.

Possibilities for cross reference

This area defines how the parameter can be cross referenced to other parameters to pursue business analysis.
